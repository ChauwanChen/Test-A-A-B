# teste A/A/B

Verificar a diferença estatística dos grupos sobre a alteração de toda a fonte do aplicativo.

Os usuários são divididos em três grupos: dois grupos de controle recebem as fontes antigas e um grupo de teste recebe as novas. Descubra qual conjunto de fontes produz melhores resultados.

Verificar a semelhança entre os dois grupos A de controle e analisar a precisão do teste baseado em diferenças significativas e também nos informa de quanto tempo e dados precisaremos ao executar outros testes.

1. Preparo de dados para analise.
2. Estudo e verificação dos dados.
3. Estudo do funil de eventos.
4. Estudo dos resultados do experimento.
